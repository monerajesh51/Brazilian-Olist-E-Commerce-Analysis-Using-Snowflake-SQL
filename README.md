# Brazilian-Olist-E-Commerce-Analysis-Using-Snowflake-SQL

## Overview

This project analyzes the Brazilian E-Commerce Public Dataset by Olist using Snowflake SQL. The goal was to uncover insights related to sales performance, customer behavior, product trends, seller performance, and operational efficiency.

## Dataset

- 100K+ Orders
- Customers
- Payments
- Products
- Sellers
- Reviews

## Technologies

- Snowflake
- SQL

## Key Analyses

- Revenue Analysis
- Monthly Sales Trends
- Customer Lifetime Value (CLV)
- Top Customers
- Product Category Performance
- Seller Performance
- Delivery Performance
- Order Status Analysis

## SQL Concepts Used

- INNER JOIN
- GROUP BY
- HAVING
- Window Functions
- RANK()
- SUM() OVER()
- DATE_TRUNC()
- DATEDIFF()

## Business Insights

- Identified highest revenue-generating categories.
- Measured customer lifetime value.
- Evaluated seller contribution to revenue.
- Analyzed delivery efficiency and customer satisfaction.
- Tracked monthly sales performance.
